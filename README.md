# WeatherForecast

 WeatherForecast is a  App where you can check weather and forecast of any location

 [Visit Now](https://weathers-forecast.netlify.app/) 


## Installation

```bash
  $ git clone https://github.com/Roushan-Raj-au9/weather-app.git
  $ cd weather-app
  $ npm install
  $ npm start
```


## Built with

- React JS
- React Context
- React Bootstarp
- Axios
- Openweathermap Api
